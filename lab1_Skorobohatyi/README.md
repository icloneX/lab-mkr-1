ТЕМА: Встановлення інструментів та створення, відлагодження та виконання проєктів у інтегрованому середовищі PSoC® CreatorTM
МЕТА: ознайомитися з функціональними можливостями інтегрованого середовища Logic та PSoC® CreatorTM 4.4, навчитися створювати, відлагоджувати та виконувати проекти, використовуючи засоби інтегрованого середовища PSoC® CreatorTM.
Завдання:
1) Встановити програмне забезпечення: середовище розробки PSoC Creator та логічний аналізатор Logic.
2) Створити новий проєкт: обрати пристрій CY8C4245AXI-483 та шаблон Empty schematic.
3) Додати на схему піни: перетягнути з каталогу Ports and Pins компоненти Digital Input Pin та Digital Output Pin.
4) Налаштувати кнопку (Input): подвійним кліком задати ім'я Button, режим роботи Resistive pull up та початковий стан High (1).
5) Налаштувати світлодіод (Output): задати ім'я LED, режим роботи Strong drive та початковий стан Low (0).
6) Призначити фізичні піни: у вкладці Pins (файл .cydwr) підключити Button до порту P0[7], а LED до одного з кольорів (P1[6] — червоний, P0[2] — зелений, або P0[3] — синій).
7) Перегенерувати проєкт: натиснути Build → Build Project (Shift+F6), щоб створилися необхідні API.
8) Написати код: відкрити main.c та додати умову if/else, щоб кнопка вмикала/вимикала світлодіод.
9) Запрограмувати плату: скомпілювати код та прошити його у PSoC 4 Pioneer Kit (Ctrl+F5).
10) Модифікувати проєкт: змінити колір діода (змінивши пін у .cydwr), знову перегенерувати та запрограмувати плату.
11) Запустити готовий приклад: через File → Code Example знайти та запрограмувати проєкт CapSense_CSD_P4_Design.
12) Сформувати звіт: додати мету, висновки, скриншот схеми .cysch, скриншот налаштування пінів .cydwr та код файлу main.c
