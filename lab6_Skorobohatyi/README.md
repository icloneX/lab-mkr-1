Мета: Розробка власного комплексного проєкту на основі інтеграції периферійних модулів (матрична клавіатура, 7-сегментний індикатор, RGB світлодіод, таймери) з використанням архітектури скінченного автомата. 

У даній лабораторній роботі було створено комплексний пристрій 
«Розумний сейф», у якому всі 5 заданих режимів (кодовий замок, таймер, 
дзеркальний текст, секундомір та рухомий рядок) об'єднані в єдиний логічний 
сценарій.
