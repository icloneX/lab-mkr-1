ТЕМА: Використання зсувних регістрів та семисегментних світлодіодних індикаторів у інтегрованому середовищі PSoC® CreatorTM

Мета: Ознайомитися з принципами роботи регістрів зсуву та семисегментних світлодіодних індикаторів у мікроконтролерних системах на базі PSoC. Набути практичних навичок створення проєктів у інтегрованому середовищі
PSoC® CreatorTM, налаштування апаратних компонентів для керування індикаторами через зсувні регістри та реалізації ефективного виведення даних.

Завдання:

1.Скопіювати проєкт: зробити копію проєкту Лабораторної роботи №3.

2.Додати на схему з каталогу Cypress три піни типу «Вихід» (Digital Output Pin) та виконати їх відповідне налаштування.

3.Підключити піни у відповідності до схеми на платі.

4.Написати код для роботи зі зсувним регістром та семисегментним індикатором (включаючи створення масиву відповідності двійкових кодів, функцій відправки даних та налаштування виводу значень у головному циклі).

5.Модифікувати код таким чином, аби на семисегментний дисплей виводився номер натиснутої кнопки матричної клавіатури.

6.Модифікувати код, додавши можливість відображення символів: A, b, C, d, E, F.

7.Сформувати звіт: додати титульний аркуш, мету, скриншоти схеми TopDesign та налаштувань .cydwr, а також лістинги коду і скриншоти роботи в консолі для всіх завдань.
